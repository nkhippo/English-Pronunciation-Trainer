---

## Quick orientation

| Layer | Role |
|-------|------|
| **Runtime (GitHub Pages)** | `index.html` + JSON/i18n/fonts loaded by the browser |
| **Production wordlist** | `wordlist_GA_a1a2_plus_phonics.json` at repo root (**4,439 words**, Jul 2026) |
| **Pipeline** | `scripts/*.py` read/write `data/pipeline/` staging JSON, merge into wordlist |
| **Batch imports** | `data/batches/phase1_m*_*.json` вҖ” Phase 1 B1 expansion sources |
| **GAS TTS** | `gas/` вҖ” Google Apps Script proxy; not loaded by static site |
| **Task history** | `docs/cursor/` вҖ” Cursor instruction + implementation reports (historical paths inside may predate reorg) |
| **Canonical specs** | `docs/PURPOSE.md`, `docs/DESIGN.md`, `docs/SPECIFICATION.md` |

**Path helper for Python:** `scripts/paths.py` defines canonical paths. Prefer importing it over hard-coded strings.

## Common pipeline commands

Run from **repo root**:

```bash
# After merging a new batch into wordlist:
python3 scripts/generate_flap_ipa.py
python3 scripts/merge_flap_candidates.py
python3 scripts/generate_respelling.py
python3 scripts/merge_respelling.py

# Export GAS batch word list (update BatchWords.gs after):
python3 scripts/export_batch_words.py

# Regenerate audit markdown:
python3 tools/gen_audit_docs.py
python3 tools/validate_i18n.py
```

Staging outputs go to `data/pipeline/`. Merge scripts write `wordlist_GA_a1a2_plus_phonics.json`.

## Phase 1 B1 expansion workflow

1. Receive `phase1_mN_400_with_gloss.json` вҶ’ save under `data/batches/`
2. Merge into wordlist (inline Python or task script)
3. Run pipeline (above)
4. Verify counts; commit wordlist + `data/pipeline/` updates
5. Add `docs/cursor/instructions/` + `docs/cursor/reports/` for the task

**Remaining B1 gap (CEFR-J):** ~389 words after M4 (M5 planned).
